Board

Board Instructions

The board is where the stimulus will be placed upon, by default there will be a board color and background color, which can be altered with the 3 sliders, you can also add an image if need be. The checkbox with the Cover Board allows the user to cover the board to allow anonymity. You also have the option to lock different camera angles during the experiment this is to restrain the experimentee from viewing at different positions.
